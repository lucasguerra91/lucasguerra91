# Lucas Guerra

**Founder & Head Brewer @ Malta Brava — I build the systems my own projects run on.**

I make beer and I make software. For a long time I thought I had to pick one. Turns out the thing I was looking for was the combination: craft beer, code, and building things my own way.

I don't write software for a company. I write it for my own projects — the POS that runs my taproom, the ERP that runs my brewery, the sites that hold it all together. If a tool doesn't exist the way I need it, I build it. AI is part of how I work now: it closed the gap between the ideas I always had and the code I couldn't always write fast enough.

This is what I'm building, in case it's useful to you.

### 🍺 Malta Brava — Craft Brewery & Taproom

*Founder & Head Brewer · Encarnación, Paraguay · [maltabrava.com](https://maltabrava.com)*

A craft brewery and taproom in a city that drinks industrial beer. We're not competing with other breweries — we're building the market from scratch. That shapes everything I build for it.

- **Taproom POS** *(private)* — Open tabs, partial payments, loyalty points, WhatsApp receipts, live kitchen display. I needed all of this without paying for SaaS I'd outgrow, so I built it as a PWA on Vue 3 + Firebase. Runs on the bar counter every weekend.
- **Brewery ERP** *(in development)* — Production, batches, and keg tracking. Kegs are the real bottleneck in a small brewery — not brewing capacity — so the system is built around knowing where every keg is and what's in it.
- **[maltabrava.com](https://github.com/lucasguerra91/malta-brava-web)** — Public site built with Astro, wired to the POS so customers can check their loyalty points.

### How I think about the stack

I pick the tool for the problem, but I keep my stack tight on purpose — I'd rather go deep on a few things than spread thin across many. Firebase for real-time and auth out of the box. Astro when it's content that has to load fast and stay simple. Vue because I'm fast in it and it gets out of my way.

`Vue 3` · `Pinia` · `Firebase / Firestore` · `Astro` · `Tailwind` · `JavaScript`
